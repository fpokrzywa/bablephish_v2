# BablePhish Portal

BablePhish Portal is an AI-powered assistant platform providing multi-assistant chat capabilities, prompt catalogs, and knowledge management.

## Features

- Multiple AI Assistants
- Prompt Catalog
- Chat Interface  
- Knowledge Base Integration
- User Management

## Built With

- React 18
- TypeScript
- Vite
- Tailwind CSS
- OpenAI API

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173 to access the portal.
